# How to run the app

1. Clone the project

```
git clone <https://github.com/papeiron/react-ecommerce-timberworld.git>
```

2. Install project dependencies

```
npm i
```

3. Start json-server

```
json-server --watch .src/data/db.json --port 3004
```

4. Run the app

```
npm run dev
```
